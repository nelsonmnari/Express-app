# Express.js API

This is a simple and modular **Express.js** API boilerplate with integrated **Swagger (OpenAPI)** documentation. Ideal for building RESTful services quickly and cleanly.

## 🚀 Features

- ⚙️ Built with [Express.js](https://expressjs.com/)
- 📄 API documentation with [Swagger UI](https://swagger.io/tools/swagger-ui/)
- 📁 Organized project structure
- 📦 Includes `package.json` for dependency management

## 🗂️ Project Structure

.
├── node_modules/
├── index.js # Main entry point for the server
├── package.json # Project metadata and dependencies
├── package-lock.json # Dependency lock file
├── swagger.yaml # OpenAPI specification for Swagger


## 📦 Installation

git clone https://github.com/your-username/your-repo-name.git
cd your-repo-name
npm install

▶️ Running the Server

node index.js
The server will start on http://localhost:3000 (or your configured port).

📘 API Documentation
After running the server, visit:


http://localhost:3000/api-docs
This will show the Swagger UI with all documented endpoints from swagger.yaml.

🧪 Testing
You can test the endpoints using:

Postman tool
